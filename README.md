# Retail Data Platform

An end-to-end Azure analytics platform that ingests multi-channel retail data into a Medallion lakehouse architecture using Azure Data Factory, Azure Data Lake, Azure SQL, and Power BI.

## Architecture

Source → ADF Pipelines → Data Lake (Bronze/Silver/Gold) → Azure SQL → Power BI

## Technologies

- Azure Data Factory
- Azure Data Lake Gen2
- Azure SQL
- Python
- Power BI
- Git / GitHub

## Project Goals

This project demonstrates how to design and implement a modern data platform capable of ingesting operational data, validating and transforming it, and delivering curated datasets for analytics and reporting.
